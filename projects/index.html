Projects

   RNS Pool - This is my current project. A car pooling application. Currently developing back-end webservice in Golang with Postgresql database. Check out the the github link to see current progress. Plans for front-end apps in Android and iOS using React Native.
   Huntr - This is a Web Application that aims to track and manage your job applications all in one place. The application is built using node.js, Express.js, Mongodb, and React. I started the Project with two friends at the North-West Hacks at UBC in 2016. For more info https://app.huntr.co. Recently the work done on the application has been done by Rennie Haylock. To view the live version https://app.huntr.co/login.
   Trippi - This is a simple web application I worked on with three other team members for my introduction to databases course at UBC. The application uses PHP, MySql, and slim. The purpose of the application was to allow users to organize trips and to join trips with friends.
   MemeEditor - This is an iOS application done using Swift. It allows the user to create different Memes, share these memes through social media, and view sent Memes through different views.
